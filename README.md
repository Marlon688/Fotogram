# Fotogram 📸

My personal photo album built with HTML, CSS and JavaScript.

## Features

- Photo gallery with thumbnail view
- Click on photo to open detail view
- Navigate through photos with arrow buttons
- Close dialog with X button, ESC key or backdrop click
- Hover effects on photos and buttons
- Responsive design

## Technologies

- HTML
- CSS
- JavaScript
